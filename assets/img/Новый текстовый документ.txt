Магазин пицц. Полноценное приложение с реализованными функциями поиска, сортировки, пагинации, фильтрации, корзины. Приложение взаимодействует с серверной частью, отправляя запросы на бэкенд. 

Приложение разработано на ReactJS. Реализованы функции фильтрации коллекций с помощью категорий, поиска коллекций, пагинации и получения коллекций через бэкэнд, отображения информации о загрузке контента.

Приложение разработано на ReactJS. Реализованы функции поиска пользователей, отправки запроса на бэкенд с помощью fetch и рендера этих данных, отображение скелетона при загрузке данных, добавления/удаления пользователей из массива приглашений.

Приложение разработано на ReactJS. Реализованы функиции получения списка курсов валют из стороннего API, конвертация одной валюты в другую (без бэкенда).
